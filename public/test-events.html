Custom Alert Testing

To test with custom videos:

  1. Place your .webm video files in the public/assets/ directory
  2. Files should be named according to the event type:
    • follow-animation.webm - For follow alerts
    • subscription-animation.webm - For subscription alerts
    • raid-animation.webm - For raid alerts
    • cheer-animation.webm - For bit donations
  3. Adjust the styling in alerts.html if needed
